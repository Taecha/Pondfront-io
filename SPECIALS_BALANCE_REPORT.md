# PondFront.io Specials Balance Report

## Current State

- Lily Barrage, Dragonfly Guard, Reed Shield, and Current Push exist.
- Specials use server-side cost, cooldown, target validation, warnings, and impact rules.
- Dragonfly Guard and Reed Shield provide counterplay.

## Changes In This Pass

- Action deltas now carry special state after commands, reducing full-map resyncs.
- Final Tide does not buff specials directly, keeping late-game specials useful but not overwhelming.

## Verification

- QA playtest covered Current Push availability, special state transport, and no server crash.
